# config.json

در این فایل ما روش تونلی که می‌خواهیم استفاده کنیم را توصیف می‌کنیم.

## ساختار کلی فایل config.json

```json title="config.json"
{
    "name": "",
    "author": "",
    "config-version": 1,
    "core-minimum-version": 1,
    "nodes": []
}
```
- پارامتر name یک اسم دلخواه برای کانفیگ است.
- پارامتر author نام نویسنده کانفیگ را مشخص می‌کند.
- پارامتر config-version نسخه فایل کانفیگ را نشان می‌دهد.
- پارامتر core-minimum-version حداقل نسخه مورد نیاز هسته برنامه را تعیین می‌کند.
- آرایه nodes شامل پروتکل‌ها و تنظیمات تونل‌هایی است که می‌خواهیم استفاده کنیم.
